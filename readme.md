### 基于react开发的仿网易云音乐官网PC端

#### 后台采用开源项目 Node.js API service
https://github.com/Binaryify/NeteaseCloudMusicApi.git

### 
* 主要使用react + react-hooks + redux + react-router
* styledcomponents
* AntDesign组件库
* axios的二次封装

#### 项目下载
git clone https://github.com/Ayu-1019/react-music.git

#### 安装依赖
yarn install

#### 运行
yarn start
